# os-installer

สร้างมาเพื่อใช้ในการติดตั้งระบบต่างๆที่เป็น Linux บน Termux

วิธีติดตั้ง:
```
apt update
```
```
apt install git
```
```
git clone https://github.com/mantvmass/os-installer
```
```
cd os-installer
```
```
sh build.sh
```
